1. Введение

СИУБФ (Система идентификации для управления библиотечными фондами) — это система для автоматизированной маркировки документов и проведения инвентаризации библиотечных фондов с использованием штрих-кодов. Система позволяет:

  • Генерировать уникальные штрих-коды для документов
  • Печатать стикеры со штрих-кодами
  • Проводить инвентаризацию с помощью сканера штрих-кодов
  • Управлять учетными записями пользователей
  • Настраивать интерфейс под индивидуальные потребности
Системные требования: Для работы с СИУБФ требуется современный веб-браузер (Chrome, Firefox, Edge, Safari) и подключение к интернету.

2. Авторизация в системе

Для доступа к системе необходимо авторизоваться с помощью учетных данных.

  1. Откройте страницу авторизации
  2. Введите ваш логин в поле "Логин"
  3. Введите пароль в поле "Пароль"
  4. Нажмите кнопку "Войти"
Важно: Не передавайте свои учетные данные третьим лицам. В случае утери пароля обратитесь к администратору системы.

После успешной авторизации вы будете перенаправлены на главную страницу системы.

4. Маркировка документов

Модуль маркировки позволяет создавать и печатать штрих-коды для документов. Доступно два режима работы:

Автоматическое формирование

Система генерирует последовательность штрих-кодов по заданным параметрам:

  • Начальный код
  • Конечный код
  • Количество штрих-кодов
  • Количество страниц

Ручное формирование

Возможность ввода отдельных инвентарных номеров для печати:

  • Ввод номеров вручную
  • Проверка на дубликаты
  • Удаление ошибочных номеров
  • Печать до 40 номеров на листе

4.1 Автоматическая маркировка

Для автоматического формирования штрих-кодов:

  1. Выберите библиотеку из списка (если доступно)
  2. Система автоматически заполнит начальный код и количество
  3. При необходимости измените параметры:
    • Начальный код (7 цифр)
    • Конечный код (рассчитывается автоматически)
    • Количество страниц (1 страница = 40 штрих-кодов)
  4. Нажмите "Печать штрих-кодов" для перехода к предпросмотру

4.2 Ручная маркировка

Для ручного ввода штрих-кодов:

  1. Введите инвентарный номер в поле ввода
  2. Нажмите "Добавить" или клавишу Enter
  3. При обнаружении дубликата система предложит варианты действий
  4. Для удаления номера нажмите × на соответствующей карточке
  5. Для очистки всего списка нажмите "Очистить все"
  6. Когда список готов, нажмите "Печать"
Примечание: Максимальное количество штрих-кодов на одном листе — 40. При добавлении меньшего количества на листе останутся пустые места.

4.3 Печать штрих-кодов

После формирования списка штрих-кодов:

  1. Система отобразит предпросмотр страницы печати
  2. Каждый штрих-код будет представлен в виде:
    • Графического изображения штрих-кода (Code39)
    • Текстового представления номера
  3. Проверьте корректность данных
  4. Нажмите "Печать" для отправки на принтер
  5. Используйте "Назад" для возврата к редактированию
Рекомендация: Перед массовой печатью сделайте пробный оттиск для проверки качества и правильности штрих-кодов.

5. Инвентаризация

Модуль инвентаризации позволяет проводить учет документов с помощью сканера штрих-кодов или ручного ввода.

Режимы сканирования

Доступны два режима работы:

  • Автоматический: добавление номера сразу после сканирования (если сканер отправляет Enter)
  • Ручной: требуется нажатие кнопки "Добавить" после сканирования

Управление списком

Работа с отсканированными номерами:

  • Просмотр всех отсканированных номеров
  • Удаление ошибочных номеров (кнопка ×)
  • Очистка всего списка
  • Экспорт данных в различные форматы

Инструкция по проведению инвентаризации:

  1. Выберите режим сканирования (автоматический/ручной)
  2. Выберите базу данных, если их доступно несколько
  3. Сканируйте штрих-коды документов или вводите номера вручную
  4. Система автоматически подсчитывает количество отсканированных документов
  5. При обнаружении дубликата система предложит проверить штрих-код
  6. Для удаления номера нажмите × на соответствующей карточке
  7. По окончании инвентаризации нажмите "Экспорт" для сохранения результатов

Экспорт данных

Результаты инвентаризации можно экспортировать в трех форматах:

  1. CSV (Excel): для дальнейшей обработки в табличных редакторах
  2. Текстовый файл: простой список номеров
  3. JSON: структурированные данные для интеграции с другими системами
Совет: Для быстрой проверки документа кликните по его номеру в списке — система откроет информацию о документе в новом окне в АБИС OPAC-Global.

6. Управление персоналиями

Модуль управления пользователями доступен только администраторам системы и позволяет:

Регистрация новых пользователей

Создание учетных записей с указанием:

  • Наименования пользователя
  • Уникального идентификатора
  • Пароля

Просмотр пользователей

Управление существующими учетными записями:

  • Поиск пользователей
  • Редактирование данных
  • Блокировка/удаление

Для регистрации нового пользователя:

  1. Перейдите в раздел "Персоналии"
  2. Выберите "Зарегистрировать нового пользователя"
  3. Заполните все поля формы:
    • Наименование пользователя
    • Уникальный идентификатор
    • Пароль
  4. Нажмите "Зарегистрировать"
Важно: Администратор должен сообщить пользователю его учетные данные.

7. Настройки системы

Модуль настроек позволяет персонализировать интерфейс системы под индивидуальные потребности.

Доступные настройки:

Цветовая тема

Переключение между светлой и темной темой интерфейса

Анимация элементов

Включение/отключение анимации интерфейсных элементов

Режимы доступности

Специальные режимы для пользователей с ограниченными возможностями:

  • Высококонтрастный режим
  • Черно-белый режим

Размер текста

Увеличение/уменьшение размера шрифта интерфейса

Все изменения настроек применяются мгновенно и сохраняются для последующих сеансов работы.

Примечание: Настройки сохраняются в браузере, поэтому при смене устройства или браузера потребуется повторная настройка.
